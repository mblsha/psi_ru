Инструкции по установке русского языка в Psi

1) Скачайте файл руссификации, щелкнув по ссылке «Lang Pack» в нижней части страницы.

2) Поместите файл в нужный каталог в зависимости от вашей операционной системы:

GNU/Linux — скопируйте psi_ru.qm to $HOME/.psi/

Windows 95/98/Me — скопируйте psi_ru.qm в C:\Program Files\Psi\PsiData (если вы установили Psi в какое-то другое место, скорректируйте маршрут соответственно).

Windows 2000/XP — есть два способа:

а) скопируйте psi_ru.qm в C:\Program Files\Psi (если вы установили Psi в какое-то другое место, скорректируйте маршрут соответственно).
б) если у вас нет административных прав, скопируйте psi_ru.qm в C:\Documents and Settings\USERNAME\PsiData (не забудьте подставить реальное имя пользователя вместо USERNAME).
